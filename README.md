# Carlo-Broderick

Cursor agent rules for reproducible R and Python data-science work (environmental, ecological, geospatial).

## Usage

Copy or symlink the rule file into your Cursor rules directory:

```bash
mkdir -p ~/.cursor/rules
cp .cursor/rules/data-science-rules.mdc ~/.cursor/rules/
```

Or add this repo as a reference when configuring Cursor project rules.

## Contents

- `.cursor/rules/data-science-rules.mdc` — always-applied conventions for script structure, geospatial handling, parallel processing on shared servers, README maintenance, and agent guardrails.
